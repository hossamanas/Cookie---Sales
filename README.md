# Cookie---Sales


📌 Project Overview


This project focuses on analyzing Cookie Sales data using Microsoft Excel. The aim is to clean, organize, and extract valuable insights from sales data to better understand business performance and customer behavior.



🎯 Objectives


• Clean and preprocess raw sales data
• Analyze overall sales performance
• Identify best-selling products
• Track revenue trends and patterns



🛠️ Tools Used


• Microsoft Excel
• Power Query (for data cleaning & transformation)
• Pivot Tables
• Charts & Visualization



🔄 Data Cleaning Process


• Removed duplicate records
• Handled missing or inconsistent values
• Standardized date and currency formats
• Transformed and structured data using Power Query



📈 Key Insights


• Top-selling cookie types based on revenue
• Best-performing time periods (daily/monthly trends)
• Sales distribution across different categories
• Customer purchasing patterns



📊 Dashboard / Visualization


• Interactive sales dashboard
• Revenue and quantity breakdown
• Product performance comparison
• Key KPIs (Total Sales, Profit, Top Product)



🚀 Skills Demonstrated


• Data Cleaning
• Data Analysis
• Data Visualization
• Business Insight Extraction
• Attention to Detail



💡 Conclusion


This project enhanced data analysis skills by transforming raw cookie sales data into meaningful insights, helping to better understand sales trends and support data-driven decision-making.
